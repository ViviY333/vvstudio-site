Surge — First AI agent launchpad on Sui

2025.8-2026
Overview

Surge is the first token launchpad built natively on Sui blockchain, designed to simplify how creators deploy and manage AI agents and crypto projects.

Challenge

How to make blockchain deployment feel as simple as filling out a form?
How to guide first-time creators through technical decisions without overwhelming them?
How to build trust in a new launchpad when users are custodying real assets?

Project List

A clean discovery page where users browse live token sales—filter by status, sort by momentum, and see real-time countdowns or participation buttons at a glance.

Key Screens

Trading Dashboard—Full project detail view combining price charts, milestone tracking with visual river progress, and integrated swap—everything needed to research, monitor, and trade without leaving the page.

Outcome

First launchpad live on Sui mainnet. 500+ agents created in first month. Community reached 24.4k followers.

Takeaway

For Web3 tools, abstraction is trust. Every technical detail we hid behind smart defaults was a barrier removed. Users don't need to understand the chain—they need to feel confident their project will launch correctly.
